## Income calculator

usage: 

```bash
./income_calc -a <amount> -l <location> [-m married]
```
-a - salary offered without currency symbol

-l - Tax residence: 
         
         UK, London
         NY, NYC - cannot do New York (separated by a space in terminal)
         Cali, California
         Seattle

-s - stock options 

```-s <amount_of_shares> <stock_quote>```
